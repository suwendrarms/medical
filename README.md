## Run Locally

Clone the project

```bash
   git clone https://github.com/suwendrarms/php-exam.git
```

Go to the project directory

```bash
  cd [your location]
```

Install dependencies

```bash
  composer Install
  copy .env.example .env
  php artisan key:generate
```

Start the server

```bash
  php artisan serve

```

Database script path

```bash
  Sql_script/database_sample.sql

```
